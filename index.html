Web Development: ESLFiles.com

Working on ESLFiles.com, I deleveped a file download system to allow users of the site to download files based on their permissions and integrated into the Wordpress CMS. This system includes a PHP/MySQL backend for admin users to upload files, track downloads, and analyze additional information on the files being served.

I also created a file template creator with PHP that allows users to upload images and generate dynamic PDF files with those images. The PDF files with the formatted images can then be saved and downloaded.

Web Development: Archeage.mmodex.com

Working on Archeage.mmodex.com, using HTML, CSS, Javascript, jQuery with PHP & MySQL, I created web-based tools to help players get more out of playing this MMORPG game. Some tools include:

  • Enchanting Simulator to allow players to simulate upgrading their equipment with game-like conditions without the risk of actually losing their equipment in game.
  • Trade Calcuator that helps players optimze their gold profits from doing specific trading quests in game.
  • Skill Calculator for players to calcuate and optimze their skill points for their character.
